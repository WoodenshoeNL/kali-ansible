## Windows

#### Create window
```bash
Ctrl + b - c
```

#### Move to next Window
```bash
Ctrl + b - n
```

#### Move to Previous Window
```bash
Ctrl + b - p
```

#### Kill Window
```bash
Ctrl + b - &
```

#### Rename Window
```bash
Ctrl + b - ,
```
