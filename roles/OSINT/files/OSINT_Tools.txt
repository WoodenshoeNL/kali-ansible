================================================================================
                           OSINT Tools Reference
================================================================================

jq
--
  Usage: jq '<filter>' <file.json> | cat file.json | jq '<filter>'
  Description: Lightweight command-line JSON processor for parsing and transforming JSON data.
  OPSEC: SAFE - Local tool, processes data offline.

ripgrep (rg)
------------
  Usage: rg <pattern> <path>
  Description: Blazingly fast recursive grep tool, respects .gitignore by default.
  OPSEC: SAFE - Local tool, searches files offline.

Exiftool
--------
  Usage: exiftool <file> | exiftool -all= <file> (strip metadata)
  Description: Read, write and edit meta information in image, audio and video files.
  OPSEC: SAFE - Local tool, processes files offline.
  Website: https://exiftool.org/

Sherlock
--------
  Usage: sherlock <username>
  Description: Hunt down social media accounts by username across social networks.
  OPSEC: NOT SAFE - Makes direct requests to social media sites, reveals your IP.
  GitHub: https://github.com/sherlock-project/sherlock

Maigret
-------
  Usage: maigret <username>
  Description: Collect a dossier on a person by username from thousands of sites.
  OPSEC: NOT SAFE - Makes direct requests to thousands of sites, reveals your IP.
  GitHub: https://github.com/soxoj/maigret

Sublist3r
---------
  Usage: sublist3r -d <domain>
  Description: Fast subdomain enumeration tool using OSINT sources.
  OPSEC: PARTIALLY SAFE - Uses search engines (passive), but DNS queries may reveal interest.
  GitHub: https://github.com/aboul3la/Sublist3r

TheHarvester
------------
  Usage: theHarvester -d <domain> -b all
  Description: E-mails, subdomains and names harvester from public sources.
  OPSEC: PARTIALLY SAFE - Most sources are passive (search engines), some modules connect directly.
  GitHub: https://github.com/laramies/theHarvester

Carbon14
--------
  Location: ~/Programs/Carbon14
  Usage: cd ~/Programs/Carbon14 && source venv/bin/activate && python carbon14.py <URL>
  Description: Estimate the date of a web page by analyzing metadata and resources.
  OPSEC: NOT SAFE - Makes direct requests to target URL, reveals your IP.
  GitHub: https://github.com/Lazza/Carbon14

Recon-ng
--------
  Location: ~/Programs/recon-ng
  Usage: cd ~/Programs/recon-ng && source venv/bin/activate && python recon-ng
  Description: Full-featured web reconnaissance framework with modular design.
  OPSEC: VARIES - Depends on modules used; some passive, some active. Check module docs.
  GitHub: https://github.com/lanmaster53/recon-ng

Spiderfoot
----------
  Location: ~/Programs/spiderfoot
  Usage: cd ~/Programs/spiderfoot && source venv/bin/activate && python sf.py -l 127.0.0.1:5001
  Description: Automated OSINT collection with web interface; integrates 200+ data sources.
  OPSEC: VARIES - Many passive modules, but some query targets directly. Review module settings.
  GitHub: https://github.com/smicallef/spiderfoot

Change Detection (changedetection.io)
--------------------------------------
  Usage: changedetection.io
  Description: Self-hosted website change detection, monitoring and notification service.
  OPSEC: NOT SAFE - Makes direct requests to monitored URLs, reveals your IP.
  GitHub: https://github.com/dgtlmoon/changedetection.io

Blackbird
---------
  Location: ~/Programs/blackbird
  Usage: cd ~/Programs/blackbird && source venv/bin/activate && python blackbird.py -u <username>
  Description: OSINT tool to search for accounts by username across social networks.
  OPSEC: NOT SAFE - Makes direct requests to social media sites, reveals your IP.
  GitHub: https://github.com/p1ngul1n0/blackbird

WhatsMyName-Python
------------------
  Location: ~/Programs/WhatsMyName-Python
  Usage: cd ~/Programs/WhatsMyName-Python && source venv/bin/activate && python whatsmyname.py -u <username>
  Description: Username enumeration tool based on WhatsMyName project data.
  OPSEC: NOT SAFE - Makes direct requests to websites, reveals your IP.
  GitHub: https://github.com/C3n7ral051nt4g3ncy/WhatsMyName-Python
